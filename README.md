### About the App
DuctuAI is a mobile app designed to help users, particularly seniors, navigate and use smartphones with confidence. It combines AI-powered chat assistance, step-by-step animated tutorials, and audio explanations to make learning intuitive and accessible. The app also includes interactive quizzes to reinforce learning, empowering users to become independent and confident in using their devices.
